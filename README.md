# addmem v1.0

**addmem** es un addon de monitoreo de rendimiento y comportamiento para **World of Warcraft 3.3.5a (WotLK)**, optimizado para el servidor **NaerZone**. Permite detectar en tiempo real el consumo de memoria, picos de lag y comportamientos maliciosos de otros addons.

---

## 🚀 Funcionalidades y Alertas

| Alerta/Función | Descripción |
|---|---|
| **Memoria** | Visualiza el consumo exacto de RAM de cada addon activo en tu interfaz |
| **Picos de Lag** | Detecta addons que bloquean el juego temporalmente (requiere perfilado CPU) |
| **Fuga de Memoria** | Registra addons cuyo consumo no deja de crecer descontroladamente |
| **Spam Oculto** | Detecta addons que saturan el canal AddonMessage provocando desconexiones |
| **Robo de Foco** | Avisa si un addon intenta tomar el control del chat sin tu permiso |
| **Inyección** | Bloquea y alerta si un addon intenta ejecutar comandos o macros maliciosas |
| **Segundo Plano** | Modo de monitoreo silencioso sin necesidad de mantener la ventana abierta |
| **Check Versión** | Consulta silenciosamente qué versión de addmem usa el resto de tu banda/grupo |

> **Historial Persistente**: Las alertas críticas de errores o fugas se guardan entre sesiones para que puedas analizarlas al día siguiente.

---

## 🛠️ Instalación

1. Copia la carpeta `addmem` en `World of Warcraft\Interface\Addons\`
2. Asegúrate que la carpeta se llame exactamente `addmem`
3. Actívalo en el menú de Addons al iniciar el juego

---

## 📖 Uso Rápido

- Usa el comando **`/addmem`** o haz clic izquierdo en el botón del minimapa para abrir la ventana
- Clic derecho en el minimapa para **activar/desactivar** el monitoreo rápidamente
